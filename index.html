Glosario de Programación

Definiciones breves para entender los pilares del código.

1. Algoritmos

Es una serie de pasos ordenados y lógicos que se siguen para resolver un problema o cumplir un objetivo.

2. Variables

Son contenedores o espacios de memoria reservados para guardar datos (números, texto, etc.) que el programa puede usar y cambiar más adelante.

3. Funciones

Bloques de código reutilizables creados para realizar una tarea específica. Evitan tener que escribir las mismas instrucciones una y otra vez.

4. Condicionales

Estructuras que permiten al programa tomar decisiones. Ejecutan un fragmento de código u otro dependiendo de si se cumple una condición (ej: "Si la contraseña es correcta, ingresar").

5. Bucles (Loops)

Instrucciones que repiten un bloque de código varias veces mientras se cumpla una condición determinada, ahorrando tiempo y líneas de desarrollo.
